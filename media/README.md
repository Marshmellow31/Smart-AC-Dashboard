# Media

Drop the hardware build photos here so they render in the main README.

The main README references these filenames — add them with these exact names
(or update the links in `README.md` to match yours):

| Filename | Suggested shot |
|----------|----------------|
| `parts.jpg` | All the parts laid out — ESP32, IR transmitter, jumpers, resistors, transistors |
| `wiring-direct.jpg` | The working direct wiring (IR transmitter straight to the ESP32) |
| `wiring-transistor.jpg` | The transistor-driver wiring (for longer range) |
| `breadboard.jpg` | The assembled breadboard / final build |
| `webapp.jpg` or `webapp.png` | Screenshot of the web app running on your phone |
| `in-action.jpg` | The device mounted and pointing at the AC |

Phone photos are usually huge — resize to ~1600 px wide and keep each file
under ~500 KB so the repo stays light and the README loads fast.
